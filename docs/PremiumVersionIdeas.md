# Premium Features
For the paid/pro version of the app, or unlockable with in-app purchase

## Specific Commander Tools

### Brudiclad
* Turn all tokens the user has into a copy of one of the tokens. 
* Those tokens retain their current counters, tapped, untapped status, etc.

### Krenko
* Create tokens according to krenko's power, keep track of number of goblins, 

### Chatterfang
* Special controls to turn chatterfang on or off, when adding new tokens make the extra squirrel. etc.

## Turn on Token Modifying Cards
* Get a list of doublers or other modifiers that the user can enable/disable. Turning them on automatically adjusts the multiplier setting to make the correct tokens.
    * Eg. Chatterfang, Academy Manufactor,

### Rhys
* Global doubling mechanic that creates a copy of every existing token (without the counters).
