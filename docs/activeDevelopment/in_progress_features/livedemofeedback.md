# Live Demo Feedback — Decks Feature

Collected during user demo session, March 2026.

## Improvements / Feature Requests

(Collecting items below as they come in)

1. **Custom tokens in Recents + persistent custom token library**
   - Custom-created tokens should appear in the "Recent" tab on token search screens
   - Expand this into a dedicated "Custom" section/tab in token search
   - Automatically save user-created custom tokens to a persistent custom token database so they can be reused in future sessions

2. **Artwork upload during custom token creation**
   - When creating a custom token (from search menu or adding to a deck), there's no way to set artwork inline
   - Users should be able to optionally upload/select art as part of the creation flow, rather than having to edit the token after the fact

3. **+1/+0 and +0/+1 counters**
   - New counter types that modify power or toughness independently
   - Must affect displayed P/T like +1/+1 counters do
   - Do NOT auto-cancel with each other or with +1/+1 and -1/-1 counters
